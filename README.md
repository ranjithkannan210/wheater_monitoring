# Weather Monitoring System

## 📌 Project Overview

The Weather Monitoring System is a web-based application designed to monitor and display weather information. It helps users check weather conditions such as temperature, humidity, wind speed, and other weather details.

## ✨ Features

- View current weather conditions
- Display temperature information
- Monitor humidity levels
- Check wind speed
- Search weather by location
- Simple and user-friendly interface
- Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
